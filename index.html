Yoga App

Developing a user-friendly yoga app has been an exciting journey that allowed me to combine my passion for wellness with my skills in front-end development and UX/UI design. This app offers guided yoga sessions tailored to users of all fitness levels, from beginners to advanced practitioners. By providing personalized routines and progress tracking features, the app encourages consistent practice and helps users stay motivated on their wellness journey.

The design of the app emphasizes simplicity and calmness, with a clean interface that makes navigation intuitive and stress-free. I focused on creating a soothing visual experience that aligns with the peaceful nature of yoga, using soft colors and minimalistic layouts. This attention to detail not only improves usability but also enhances the overall user experience. Through this project, I have showcased my ability to develop responsive and mobile-friendly applications that meet real user needs.

Weather App

I developed a basic weather application using React that provides real-time weather information based on user input. The app retrieves data from a public API and displays key details such as temperature, humidity, wind speed, and weather conditions in a clear, user-friendly interface.

Leveraging React's component-based structure and hooks like useState and useEffect, I ensured efficient data fetching and state management. The design emphasizes simplicity and responsiveness, making it accessible across devices. This project highlights my ability to build interactive web applications, integrate APIs, and create clean, functional user experiences.
